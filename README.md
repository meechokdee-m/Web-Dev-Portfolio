# Personal Portfolio Website

A responsive personal portfolio website built to present my profile,
skills, and engineering projects in a clean and structured format.

## Features
- Multi-page layout (Home, Experience, Projects, Contact)
- Responsive design for desktop and mobile
- Dark / Light mode toggle
- Simple and readable UI

## Tech Stack
- **Languages**: HTML, CSS, JavaScript
- **Styling**: CSS Variables, Responsive Layout
- **Functionality**: JavaScript (DOM manipulation, localStorage)
- **Tools**: Git, GitHub, VS Code

## Structure
- `index.html` – Profile and technical overview  
- `experience.html` – Academic and internship experience  
- `projects.html` – Project showcase  
- `contact.html` – Contact information  

## Purpose
This website was created to practice frontend web development fundamentals
and to serve as an online portfolio for job applications.
